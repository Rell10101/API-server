# API сервер

## Структура приложения
### Папки
app - в ней хранятся модули приложения\
models - обученная модель линейной регрессии и тренировочные данные
### Служебные файлы
.env - хранит путь к файлу модели\
requirements_service.txt - список требуемых библиотек для виртуального окружения

## Как запустить приложение
В терминале нужно ввести команду "uvicorn server:app --reload --port 8880"\
server - название файла\
app - название объекта FastAPI\
--reload - перезапускать сервер если исходники изменились\
--port 8880 - указание порта

## Как проверить: базовую работоспособность приложения, получить предсказание
На странице /docs можно посмотреть документацию и там же сформировать запросы по каждому эндпоинту

## Как посмотреть документацию к API
После запуска нужно зайти по ссылке http://127.0.0.1:номер_порта/docs
